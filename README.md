## 📈 Airbnb Performance Dashboard

✨ <u>About this project</u>

**1. Objective** <br>
The objective of this project was to "craft a compelling visual narrative that highlights the key features such as listing performance, property attractiveness based on the rating scores and seasonality trend”. <br>

**2. Process** <br>
Airbnb dataset includes over 250k listings in 10 major cities, including information about hosts, pricing, location, and room type, along with over 5 million historical reviews. I noticed there were some missing values and data inconsistency, so I removed them for analysis. I used ETL framework (Extract, Transform, Load), Data analysis (using DAX), and Visualization skills to identify any significant differences between cities, evaluate the cities which offer the best value for travel and identify trends or seasonality. I also applied some useful visualization techniques and concepts I learned from online learning into this project. <br>

**3. Output** <br>
The output was an interactive dashboard, which provides valuable insights for both new and experienced Airbnb hosts to assist them in making informed decisions about where and how to list their properties to maximize profitability. <br>

**4. Key Analysis Questions** <br>
* Any major differences in the Airbnb market between cities?
*	Which attributes have the biggest influence in price?
* Are you able to identify any trends or seasonality in the review data?
* Which city offers a better value for travel? <br>

**5. Data source** <br>
Please download Dataset here - https://mavenanalytics.io/data-playground/airbnb-listings-reviews <br>
